# gpa-scraper
A google photos album scraper

# Description
gpa-scraper allow us to get basic photos information from a shared album such:
- public url
- filename
- description
- file size
- image size
- creation date
    
# Usage
./index.js --url=https://photos.app.goo.gl/gZ1jq1L16cTenTwT7 --repo=sqlite --filename=./data/animals.db
